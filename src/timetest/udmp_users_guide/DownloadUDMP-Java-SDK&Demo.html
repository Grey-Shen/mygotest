Changelog

  1. 将 Gson 改成内部依赖。
  2. Bizdata 增加 null 检查。key 或者 value 如果为 null 会抛出异常。此改动为 不兼容 改动！涉及 API：
    • Document#putIntoBizdata
    • Document#getFromBizdata
    • Document#delFromBizdata
    • Page#putIntoBizdata
    • Page#getFromBizdata
    • Page#delFromBizdata

  1. 增加 Document#update() 方法，允许更新已有 Document 对象的 Bizdata。

  • UDMP Java SDK v1.3.0
  • UDMP Java SDK Demo v1.3.0

v1.2.0 2017-02-15

Changelog

  1. 移除 Document 对象的 title 字段以及相关方法。涉及 API 变动：
    • Document#init() 方法不再需要 title 参数。
    • Document#getTitle() 方法被移除。
  2. Page 对象必须依赖 Document 对象存在，即只能通过一个 Document 实例上的实例方法构造新的 Page 实例。涉及 API 改动：
    • Page#init 被移除，必须使用 Document#newPage() 方法作为替代。具体用法参考 demo。
    • Document#appendPage() 方法不再需要 Page 实例参数。
    • Page#setContent() 方法被移除，必须在 Document#newPage() 时传入 Page 的非结构化数据内容。具体用法参考 demo。
  3. Document#getDocId() 重命名为 Document#getId()。
