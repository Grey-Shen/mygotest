3.4 文档查询

文档查询

业务系统也可以使用一个或者多个 Bizdata 字段查询 UDMP，批量获得多个文档。本功能需要应用对操作数据拥有 Read 权限。

简单查询

int limit = 20;
int offset = 0;
DocumentQuery conditions = new DocumentQuery();

conditions.docBizdata("transaction_no", "12478912738");

DocumentQueryResult result = Document.query(conditions, limit, offset);

可以通过 DocumentQuery 对象上的 .docBizdata() 方法设置作为查询条件的结构化业务数据字段名称和值。

高级查询

也可以多次调用 .docBizdata() 该方法，或者 .pageBizdata() 等其它类似方法设置多个查询条件，多个查询条件在逻辑上是 AND 关系。还可以通过 .sortBy 方法设置查询结果的排序条件。

conditions.docBizdata("transaction_no", "12478912738")
          .pageBizdata("customer_name", null)
          .operator(OperatorField.CREATE_BY, "chenzhichun")
          .sortBy(
              DocumentQuery.FieldCategory.DOC_BIZDATA,
              "transaction_time",
              DocumentQuery.SortType.ASC
          );

请查阅 UDMP Java SDK 参考文档了解 DocumentQuery 的详细用法。
