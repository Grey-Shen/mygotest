术语表

UDMP

Unstructured Data Management Platform 非结构化数据管理平台。

PabOS

Ping An Bank Object Storage 平安银行对象存储。

App

在 UDMP 中，接入的业务系统都称为应用。业务系统在接入之前需要通过流程申请注册应用。

AppId

用于在 UDMP 中唯一区分应用的标识符，它是一个长度为 8 字节的字符串，在注册应用时由 UDMP 自动生成。

AppKey

与 AppId 一起用于业务系统向 UDMP 证明自己身份的重要信息，作用与密码相同。因此业务系统必须确保自己 AppKey 的安全性，避免被泄漏。

Document

Java SDK 中的 Document 对象是 UDMP 组织业务数据的基本单元，参考业务数据组织方式中的详细解释。

Page

Java SDK 中的 Page 对象是直接容纳非结构化数据文件的基本对象，参考业务数据组织方式中的详细解释。

DocId

文档的 ID，用于在 UDMP 整个系统中唯一标识一个文档。它是一个长度 80 字节的字符串，在上传文档时由 UDMP API 自动生成。

PageNo

文档页的页码，用于在文档内唯一标识一个文档页。Java SDK 在上传文档时按文档页添加顺序，从 1 开始递增生成 PageNo。因此一个文档中最大的 PageNo 与文档页数量在数字上完全相同。

Bizdata

Document 或 Page 对象上的结构化业务数据字段的统称，业务系统可以自定义这些字段的名称。一个 Document 或 Page 可以有无限数量个 Bizdata 字段。
