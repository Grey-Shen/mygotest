3.3 文档获取

文档获取

上传文档后，可以通过 DocId 获取文档以及其包含的业务数据和所有文档页。本功能需要应用对操作数据拥有 Read 权限。

获取文档

String docId;
Document doc = Document.find(docId);

参数说明

  • docId 指定的文档 ID。

获取业务数据

String key;
String value = doc.getFromBizdata(key);

参数说明

  • key 需要获取值的 Bizdata 字段名称。

获取文档页（Page）

int pageSize = doc.getPageCount();
int pageNo;
Page page = doc.getPage(pageNo);
// 获取文档页的 Bizdata 字段
String value = page.getFromBizdata("key");
// 获取内容读取流
InputStream stream = page.getContent();
// 读取 Page 内容
// ...
stream.close();
// 获取下载地址
String url = page.getContentUrl().toString();

注意：toString() 后得到的 URL 字符串包含认证信息，这些 URL 仅在 toString() 方法调用后 2 小时候内有效。

参数说明

  • pageNo 页面页码，从 1 开始到 pageSize，超出范围将抛出异常。

获取所有文档页

需要访问业务数据的场景

Iterable<Page> pages = doc.getPages();

不需要访问业务数据的场景

获取所有文档页的下载地址：

List<String> urls = doc.getPageContentUrlStrings();
