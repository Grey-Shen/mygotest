2.2 申请设置防火墙策略

申请设置防火墙策略

为确保不受防火墙影响，正常访问 UDMP 的各项服务，业务系统需要申请设置防火墙策略，开通对以下服务对应 IP 地址 80 端口（TCP 协议）的访问:

测试环境

服务说明 域名 IP 地址
PabOS 下载出口 io.pabosuat.sdb.com.cn 10.14.2.27
PabOS 上传入口 up.pabosuat.sdb.com.cn 10.14.2.27
PabOS RS 服务 rspub.pabosuat.sdb.com.cn 10.14.2.27
PabOS RSF 服务 rsf.pabosuat.sdb.com.cn 10.14.2.27
PabOS API 主机 apiserver.pabosuat.sdb.com.cn 10.14.2.27
UDMP API api.udmpuat1.sdb.com.cn 10.14.210.134
UDMP Web Portal portal.udmpuat1.sdb.com.cn 10.14.211.157

测试环境 PabOS 的几个服务使用相同的 IP 地址，因此一共只需要申请 3 个 IP 地址的防火墙规则。
