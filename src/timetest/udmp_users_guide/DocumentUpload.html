3.2 文档上传

文档上传

初始化配置后，可以将非结构化的业务数据以 Document 的形式组织上传到配置指定的应用。 本功能需要该应用对操作数据拥有 Read 和 Write 权限。

初始化 Document

将 Document 上传到配置指定的应用：

String operator;
Document doc = Document.init(operator);

或者将 Document 上传到其他有操作权限的应用：

String operator;
String targetAppId;
Document doc = Document.init(operator, targetAppId);

参数说明

  • operator 操作员信息（可以是名字，编号等）。

可选属性

  • targetAppId 上传到 UDMP 所属的应用 ID 。

附加业务数据

String key;
String value;
doc.putIntoBizdata(key, value);

参数说明

  • key 需要设置的 Bizdata 字段名称。
  • value 需要设置的 Bizdata 字段值。

注意： 如果原有键有对应的值，则该值会被新值覆盖。

附加文档页（Page）

String path;
String key;
String value;
FileInputStream stream = new FileInputStream(path);
Page page = doc.newPage(stream)
               .putIntoBizdata(key, value);

参数说明

  • path 文档页的内容路径。
  • stream 需要绑定的文档页内容。

上传到 UDMP

doc.save(Document.NO_PACK);

注意：建议文档页较大（小于 5MB），数量较少（小于 20 个）的文档选择 NO_PACK 方式上传。 只能对未上传文档执行保存操作。

获取文档 ID

String docId = doc.getId();

注意：保存文档 ID 到自己的业务数据库中，以便将来获取已保存的文档。
